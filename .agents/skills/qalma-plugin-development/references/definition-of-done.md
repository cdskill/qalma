# Plugin Definition Of Done

Use the applicable items. Do not add meaningless tests merely to tick a box.

## Design

- The feature is optional and selected through the consumer plugin array.
- Engine behavior and presentation are separated.
- Public names and options belong to Qalma rather than leaking upstream
  implementation details.
- The existing plugin contract is sufficient, or an architecture decision
  explicitly justifies extending it.
- The plugin stays in `libs/editor` unless a real extraction criterion exists.

## Implementation

- Plugin key is unique.
- Schema names, commands, command states, and shortcuts do not collide.
- Parse and serialization behavior are predictable.
- Configurable options have immutable defaults and early validation.
- Internal helpers remain private.
- The public barrel exports only consumer-facing symbols.
- No dead code, compatibility alias, or unrelated refactor remains.
- Repository skills are updated when the plugin changes the durable contract or
  development workflow.
- `qalma-docs-sync` has been used when the plugin changes public docs, docs
  navigation, playground behavior, or `libs/editor/README.md`.

## Consumer Experience

- Sandbox configures the plugin through `createQalmaEditor`.
- Sandbox UI is explicitly composed with consumer-owned Angular markup.
- No toolbar button or styling is generated by the editor library.
- Commands expose correct disabled and active state where applicable.
- Controls remain keyboard and screen-reader understandable.

## Verification

- Add focused unit coverage for configuration and failure modes where useful.
- Add or update browser coverage for visible behavior.
- Verify keyboard shortcuts when the plugin defines them.
- Verify parsing and serialized HTML when the document model changes.
- Run:

```bash
pnpm nx run-many -t lint
pnpm nx build sandbox
pnpm exec tsc -p libs/editor/tsconfig.lib.json --noEmit
pnpm exec tsc -p apps/sandbox/tsconfig.json --noEmit
pnpm exec tsc -p apps/sandbox-e2e/tsconfig.json --noEmit
```

- If a local server was started, stop it and verify its port is free before
  reporting completion.
